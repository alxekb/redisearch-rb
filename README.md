# redisearch-rb

A simple Ruby client for RediSearch module
http://redisearch.io/


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'redisearch-rb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install redisearch-rb
